
# Introduction to Physical AI Documentation

This documentation provides a comprehensive guide to Physical AI and Robotics, with a focus on reproducible examples and tutorials.

## What is Physical AI?

Physical AI refers to artificial intelligence systems that interact with the physical world through robotic platforms. This includes humanoid robots, autonomous vehicles, drones, and other embodied AI systems.

## Course Structure

This 13-week curriculum covers:

- **Week 1-2**: Introduction To Physical AI
- **Week 3-5**: ROS 2 Fundamentals
- **Week 6-7**: Robot Simulation with Gazebo
- **Week 8-10**: NVIDIA Isaac Platform
- **Week 11-12**: Humanoid Robot Development
- **Week 13**: Conversational Robotics

Each section includes practical, runnable examples that work on a clean Ubuntu 22.04 + ROS 2 Humble installation.
